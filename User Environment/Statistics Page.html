Statistics
Welcome to the statistics page. Click on a button below to see all graphs related to a specific input scheme. If you don't see any buttons below, you may need to refresh this page.
On the X axis is the number of tests completed, on the Y axis is both the Number of correct inputs per minute and the error rate as a percentage. Blue dots are the number of correct inputs per minute achieved in each test, and blue lines are the averages of this across all users on their Xth test. Red dots are the percentage error achieved in each test, and red lines are the averages across all users on their Xth test.
The data displayed here is updated hourly, at the 45th minute.
